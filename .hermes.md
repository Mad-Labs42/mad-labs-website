## Development

### Quick Start

```bash
# From WSL project root
cd /mnt/d/Workspaces/mad-labs-website

# Install dependencies
pnpm install

# Development server
pnpm dev

# Validate
pnpm check      # astro check (type checking)
pnpm lint       # eslint
pnpm format:check  # prettier
pnpm build      # production build
pnpm ci         # full validation suite
```

### Running Hermes on this project

```bash
cd /mnt/d/Workspaces/mad-labs-website && \
hindsight-embed -p mad-labs-website daemon start && \
hermes -s mad-labs-brand,mad-labs-website
```
